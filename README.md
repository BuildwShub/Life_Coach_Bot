# 🔥 Shub's Life Coach Bot

A personal Telegram bot that sends daily reminders and keeps you accountable.

## Daily Schedule
- 🌅 1:00 PM — Wake up
- 💪 1:30 PM — Gym
- 📚 2:30 PM — Study block
- 🐙 3:30 PM — GitHub commit
- 💼 3:50 PM — Work prep
- 😴 3:30 AM — Sleep

## Commands
- `/start` — Start the bot & get your Chat ID
- `/status` — What you should be doing right now
- `/checkin` — Log today's wins
- `/linkedin` — Generate a LinkedIn post
- `/project` — Get next data science project idea
